# Workspace

Workspace is a centralized platform for managing employees, teams, projects, and tasks within an organization.

## Features

* User Authentication
* Role-Based Access Control
* Employee Management
* Team Management
* Project Management
* Task Management
* Task Assignment
* Task Status Tracking
* Employee Profiles
* Team Member Management
* Project Progress Tracking
* Dashboard Overview

## Tech Stack

* Java
* Spring Boot
* Spring Security
* Spring Data JPA
* Hibernate
* MySQL
* Thymeleaf
* Maven
